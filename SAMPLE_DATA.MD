## How to Use This Data

### Option 1: Manual Entry (Learning)
1. Create your "Leads Database" sheet with headers in row 1
2. Manually type in a few records (1-3) to learn the process
3. Useful for understanding the workflow

### Option 2: Copy/Paste (Faster)
1. Create your "Leads Database" sheet and headers
2. Copy the data table above (rows with "|" separators)
3. Paste into your sheet starting at A2
4. Clean up formatting as needed

### Option 3: Use as Template
Use the fields and data types to understand what kinds of information belong in each column.

## Customizing Sample Data

**Want to test a specific scenario?**

### Test: "Many leads missing docs"
Add more rows with Missing Docs Flag = TRUE

### Test: "High human review volume"
Set Human Review Flag = TRUE for several leads

### Test: "Old pending docs"
Change Date Received to earlier dates (e.g., 3/20/2026) for leads with Missing Docs = TRUE

### Test: "Completed leads"
Set Intake Status = "Completed" for several leads (they'll disappear from most tables)

## Tips for Testing

1. **Start simple** — Enter just KSA-001 first, verify KPI shows 1 lead
2. **Add variety** — Add leads with different asset types one at a time
3. **Test filtering** — Add missing docs, verify exception table updates
4. **Test dates** — Create recent and old leads, verify date-based metrics
5. **Test flags** — Add human review flags, verify queue populates
6. **Test formulas** — Click on dashboard formulas to see they reference database correctly
